# Local Secrets

Эта папка нужна только для локальных ключей. Реальные ключи нельзя коммитить.

Для серверного распознавания голоса откройте `groq.env` и вставьте Groq API key:

```env
GROQ_API_KEY=gsk_your_key_here
```

Важно:

- это ключ Groq Whisper, не xAI Grok;
- OpenAI-ключи в этом проекте не используются;
- после вставки ключа нужно перезапустить API;
- без ключа основная кнопка `Диктовать` всё равно работает через браузер, но длинная серверная запись не даст текст.

Проверка после перезапуска:

```powershell
.\scripts\check-groq-speech.ps1
```

В проверке должно быть:

- `Ключ виден серверу: True`;
- `Длинная запись доступна: True`.

Если ключ есть в файле, но сервер его не видит, перезапустите API и повторите проверку.
