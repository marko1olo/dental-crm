# Клиническое Руководство Пользователя DENTE

Добро пожаловать в DENTE — современную облачную платформу для управления стоматологической практикой и диагностическим планированием. Это руководство предназначено для врачей, ассистентов и администраторов.

## 1. Загрузка и просмотр DICOM (3D КЛКТ)
Система поддерживает работу с полноценными 3D DICOM-исследованиями (КЛКТ томограммы).
- **Как загрузить**: Перейдите в ЭМК Пациента -> Вкладка "Снимки". Нажмите "Загрузить DICOM" или перетащите папку с файлами `*.dcm`.
- **Как избежать WebGL крешей (WebGL Context Lost)**: КЛКТ снимки потребляют до 1.5 ГБ VRAM. Не открывайте несколько вкладок с 3D-вьюером одновременно. Если бразуер "упал", система автоматически очистит кэш. Вы также можете принудительно очистить кэш, обновив страницу с зажатой клавишей `Shift`.
- **Пресеты Window/Level (W/L)**: 
  - Нажмите и удерживайте **Правую кнопку мыши (ПКМ)** во время движения вверх/вниз для регулировки Контраста (Window) и Яркости (Level).
  - Используйте готовые пресеты на панели MPR (Bone, Soft Tissue) для быстрой смены отображения плотных структур.

## 2. Планирование имплантации (Surgical Guide)
- **Коллизии и правила безопасности**: DENTE автоматически проверяет расстояние от имплантата до размеченного нижнечелюстного канала (IAN). Безопасным считается расстояние > 2мм.
- **Предупреждение коаксиального расхождения**: Если оси двух соседних имплантатов расходятся более чем на 15°, система выдаст предупреждение. Это критично для будущего ортопедического этапа (сложности с установкой единой мостовидной конструкции).
- **Постановка имплантата**: Удерживайте `Shift` и кликните левой кнопкой мыши на кросс-секции, чтобы установить виртуальный имплантат.

## 3. Оценка плотности кости по Misch (D1-D5)
При планировании имплантации система анализирует плотность кости (в единицах Хаунсфилда, HU) вдоль виртуальной оси сверления.
- **D1 (>1250 HU)**: Очень плотная кость. Высокая первичная стабильность, но повышен риск перегрева при сверлении (остеонекроз).
- **D2 (850 - 1250 HU)**: Оптимальная плотная трабекулярная кость. Идеально для остеоинтеграции.
- **D3 (350 - 850 HU)**: Рыхлая кость. Требует осторожного сверления и может потребовать остеоконденсации.
- **D4 (150 - 350 HU)**: Очень рыхлая кость. Низкая первичная стабильность, часто требует длинных имплантатов.
- **D5 (<150 HU)**: Незрелая кость. Требует синус-лифтинга или НКР (направленной костной регенерации).

## 4. Панорамная реконструкция (Panoramic Spline)
Для получения классической плоской панорамной картины (ОПТГ) из 3D-снимка:
1. Перейдите в Аксиальный срез (Axial View).
2. Нажмите кнопку "Построить Панораму".
3. Расставьте кликами маркеры по центру зубной дуги от левого сустава до правого.
4. Завершите двойным кликом. Вьюер развернет криволинейную поверхность и отобразит панораму.

## 5. Использование 2D Shadow Analyst
Shadow Analyst — это мощный инструмент для анализа 2D рентгенограмм (прицельных снимков).
- **Фильтры**: Используйте фильтры "Edge Detection" для четкого выделения границ корней и "Invert" для лучшей видимости кариеса (в инверсии деминерализованные участки светятся белым).
- **Измерения**: Зажмите ЛКМ и протяните линию через очаг воспаления, чтобы измерить размер периапикального поражения.
- Автоматически экспортируйте результаты в Зубную формулу (Одонтограмму) одним кликом.
